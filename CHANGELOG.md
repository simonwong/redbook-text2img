一款 Markdown 转小红书图片的在线工具

## v1.2.0

- **主题视觉升级**：四个核心渐变主题改为多层 radial-gradient 叠加的 mesh-gradient 风格，对标 Stripe / Linear 的现代美学
  - 蜜光暖阳：奶油 + 蜜桃 + 杏色四点光晕，告别高饱和的刺眼黄
  - 晨雾微光：雾蓝交织薰衣草，清晨薄雾的清雅微光
  - 樱花奶霜：蜜桃粉与奶油白交融，奶霜质感的温柔粉调
  - 墨夜极光：深邃墨色配紫蓝极光，沉静而有呼吸感
- **主题命名诗化**：「暖阳渐变 / 冷调渐变 / 小红书粉 / 暗夜黑」更名为「蜜光暖阳 / 晨雾微光 / 樱花奶霜 / 墨夜极光」，更贴合小红书审美调性
- **字色降饱和**：将正文与标题色从纯黑/高饱和色调整为带温度的深咖、深海军、深酒红，提升长文阅读舒适度
- **强调色重构**：blockquote 与 code 的背景从纯白改用各主题的奶油白基调，与背景融为一体；暗色主题点缀色升级为薰衣草紫

## v1.1.0

- **全新交互设计**：从三列卡片布局升级为双面板分栏式创意工作台，编辑体验更沉浸
- **图片轮播预览**：支持左右箭头切换、底部缩略图导航条，快速浏览多张图片
- **编辑器工具栏**：新增 Markdown 格式化按钮（加粗、斜体、标题、列表、引用、代码、分割线）
- **光标段落同步**：编辑器光标移动时，预览自动跟随到对应图片段落
- **浮动样式面板**：配置器改为右侧浮动面板，不遮挡预览，支持 Escape 关闭
- **视觉化主题选择**：主题选择从下拉框升级为网格卡片，直观预览主题效果
- **移动端适配**：编辑器全屏显示，预览通过底部 Drawer 展示，小屏隐藏导航箭头
- **配色优化**：柔化亮色/暗色模式背景，减少视觉疲劳
- **键盘导航**：支持方向键左右切换图片
- **导出体验优化**：新增批量导出进度条、导出成功动画反馈
