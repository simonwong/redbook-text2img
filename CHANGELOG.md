一款 Markdown 转小红书图片的在线工具

## v1.1.0

- **全新交互设计**：从三列卡片布局升级为双面板分栏式创意工作台，编辑体验更沉浸
- **图片轮播预览**：支持左右箭头切换、底部缩略图导航条，快速浏览多张图片
- **编辑器工具栏**：新增 Markdown 格式化按钮（加粗、斜体、标题、列表、引用、代码、分割线）
- **光标段落同步**：编辑器光标移动时，预览自动跟随到对应图片段落
- **浮动样式面板**：配置器改为右侧浮动面板，不遮挡预览，支持 Escape 关闭
- **视觉化主题选择**：主题选择从下拉框升级为网格卡片，直观预览主题效果
- **移动端适配**：编辑器全屏显示，预览通过底部 Drawer 展示，小屏隐藏导航箭头
- **配色优化**：柔化亮色/暗色模式背景，减少视觉疲劳
- **键盘导航**：支持方向键左右切换图片
- **导出体验优化**：新增批量导出进度条、导出成功动画反馈

## v1.0.0

- **预览图片尺寸调整**：优化预览图片的显示尺寸
- **封面标题居中**：封面图标题不再跟随标题对齐设置，默认居中显示
- **移动端优化**：实现响应式布局，支持移动端/桌面端自适应，优化小屏幕显示
- **Logo 更新**：更新项目 Logo 和 OG 图片

## v0.2.0

- **主题优化**：优化了清新白、暗夜黑主题，新增 Apple 备忘录主题
- **配置简化**：简化样式配置门槛，支持密度、字体、标题对齐三个快捷配置

## v0.1.0

- **编辑器升级**：集成 CodeMirror Markdown 编辑器，提供更专业的编辑体验
- **黑暗模式**：支持浅色/深色/自动三种主题模式，自动跟随系统设置
- **常见问题**：新增 FAQ 页面，解答使用相关问题
- **样式优化**：全面升级 UI 组件，支持主题自适应
- **状态持久化**：主题偏好自动保存到本地存储
